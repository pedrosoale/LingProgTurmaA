E este é um parágrafo, um dos elementos mais comuns em documentos HTML.

Existem diferentes níveis de título, do 1 ao 6:

Título nível 2

Título nível 3

Título nível 4

Título nível 5
Título nível 6

É possivel enfatizar uma palavra com o elemento em, dar ainda mais ênfase com o elemento strong, ou marcar uma palavra com o elemento mark. Embora existam os elementos i itálico e b negrito, eles não são mais recomendados, pois o HTML deve focar no conteúdo e semântica, deixando a estilização para o CSS.

Também podemos demarcar código fonte com o elemento code, ou demarcar entrada de teclado com o elemento kbd.

Podemos forçar uma quebra de linha aqui
com o elemento br.

Podemos usar sub para subscrito, como em H2O, ou sup para sobrescrito em x2.

Imagens são inseridas com o elemento img.

A imagem pode ser hospedada em outro site.

O elemento a corresponde a um link para outro documento.

Também podemos abrir link em nova aba com o atributo target = "_blank": Página 2.
